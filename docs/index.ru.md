# RebelRx Docs

Добро пожаловать в RebelRx Docs — ресурс с практическими руководствами по созданию суверенных систем, защите приватности и взятию под контроль собственного здоровья.

---

## ⚠️ Основной принцип

> Если вы не контролируете свои инструменты, данные и входящие потоки, это делает кто-то другой.

RebelRx Docs создан, чтобы помочь вам вернуть этот контроль.

## 🧭 Что включает этот сайт

RebelRx Docs предоставляет тщательно отобранную коллекцию материалов:

- **Linux-гайды** → Чистые и управляемые альтернативные операционные системы для десктопа и серверов  
- **Homelab** → Самостоятельно размещаемая инфраструктура, которой вы полностью владеете  
- **Приватность** → Снижение зависимости от платформ, основанных на слежке  
- **Здоровье** → Образование с акцентом на первопричины и осознанный подход к данным
