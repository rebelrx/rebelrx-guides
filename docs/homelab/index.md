# Homelab

Guides for building a practical, privacy-conscious homelab.

## Core areas

### Services

Applications worth self-hosting, how to deploy them, and how to maintain them.

### Networking

Reverse proxies, tunnels, DNS, VPN, and remote access patterns.

### Storage

Mounting, backup design, media libraries, and archiving.

### Standards

Compose patterns, naming conventions, folder layouts, and operational habits that make a lab easier to manage.
