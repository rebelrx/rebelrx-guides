# 📚 About

RebelRx Docs is a public documentation site focused on building sovereign systems, protecting privacy, and improving long-term health through practical, real-world guidance.

This is not a generic documentation hub.

It is a curated body of work built around one core idea:

> You should understand and control the systems that impact your life.

---

## ⚠️ Core Philosophy

RebelRx is built on three principles:

- **Control over convenience**  
- **Clarity over complexity**  
- **Long-term thinking over short-term optimization**  

Across technology and health, modern systems increasingly:

- Centralize control  
- Abstract away understanding  
- Prioritize scale over individual outcomes  

RebelRx takes the opposite approach.

---

## 🧭 What This Site Provides

RebelRx Docs provides:

- Step-by-step technical guides for building private infrastructure  
- Proven configurations based on real-world use  
- Clear recommendations (not endless options)  
- Documentation of what works — and what to avoid  

!!! tip
    These guides prioritize systems you can **understand, maintain, and trust**.
