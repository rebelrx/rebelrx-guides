# 🛡️ 隐私优先的生活方式

一份实用指南，帮助你减少对大型科技公司（Google、Apple 等）的依赖，并掌控自己的数据——同时不破坏你的日常工作流程。

---

## ✨ 理念

这**不是**一份关于极端隐私或完全脱离网络（off-grid）的指南。

它关注的是迈向下一步的小胜利：

* 减少跟踪
* 掌控你的数据
* 构建一个*由你控制*的系统

> 💡 如果你没有为产品付费，那么你就是产品本身。

---

## 🧭 你将学到什么

* 为什么隐私很重要
* 应该优先替换哪些应用/服务
* 如何在不破坏工作流程的情况下安全迁移
* 一个真实世界的实践方案

---

## 🪜 如何使用本指南

按顺序阅读：

## ⚖️ 核心权衡

| 权衡       | 现实        |
| -------- | --------- |
| 便利性 ❌    | 需要更多配置    |
| “免费”服务 ❌ | 你的数据就是产品  |
| 生态锁定 ❌   | 灵活性与控制力 ✅ |

---

## 🚀 从哪里开始（快速收益）

如果你只做一件事，从这里开始：

* 切换到 Brave 浏览器
* 使用 Signal 进行消息通信
* 设置一个自托管或可信第三方密码管理器
* 切换到开源生产力工具（如 OnlyOffice 或 BentoPDF）

> 💡 仅这些改变就能显著减少跟踪和数据被挖掘的风险。

---

## 🔒 站点隐私

RebelRx Docs **不使用**跟踪 Cookie、分析工具或第三方跟踪脚本。

不会收集任何个人数据。

> 无跟踪。无监控。无数据收集。
