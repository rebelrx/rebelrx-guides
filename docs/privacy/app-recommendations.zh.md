# 🔁 基于隐私的应用替代推荐

针对常见大型科技（Big Tech）服务的高影响替代方案。

---

## 🌐 浏览器与搜索

| 替代        | 使用                                                                              | 原因                    |
| --------- | ------------------------------------------------------------------------------- | --------------------- |
| Chrome    | [Brave](https://brave.com/)                                                     | 内置广告/追踪拦截，基于 Chromium |
| Google 搜索 | [DuckDuckGo](https://duckduckgo.com/) / [Startpage](https://www.startpage.com/) | 减少跟踪                  |

> 💡 进阶：自托管 [SearxNG](https://docs.searxng.org/)，实现完全私有的搜索引擎

---

## 📧 电子邮件

| 替代    | 使用                                                                | 原因           |
| ----- | ----------------------------------------------------------------- | ------------ |
| Gmail | [Proton Mail](https://proton.me/mail) / [Tuta](https://tuta.com/) | 端到端加密，以隐私为优先 |

> 💡 提示：使用**自定义域名**以获得长期灵活性
